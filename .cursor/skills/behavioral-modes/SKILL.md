# Behavioral Modes

## Modos tipicos

- **Brainstorm / plano:** perguntar, comparar opcoes, evitar codigo ate alinhamento.
- **Implementar:** mudancas diretas, validacao ao final.
- **Revisar / debug:** evidencias, hipoteses, passos minimos.
