# Plan Writing

## Principios

- Mantenha planos curtos, diretos e verificaveis.
- Cada tarefa deve ter um resultado observavel.
- Identifique dependencias e a ordem de execucao.
- Termine com etapa de verificacao.

## Formato recomendado

```markdown
# <nome da tarefa>

## Objetivo
<1 frase>

## Tarefas
- [ ] <acao> -> Verificar: <criterio>
- [ ] <acao> -> Verificar: <criterio>

## Concluido quando
- [ ] <criterio final>
```
