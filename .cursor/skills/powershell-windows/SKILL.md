# PowerShell (Windows)

## Regras

- Saida em terminal: preferir marcadores ASCII (`[OK]`, `[FAIL]`) quando emojis quebrarem encoding.
- Condicoes compostas com parenteses claros: `if ((A) -and (B))`.
