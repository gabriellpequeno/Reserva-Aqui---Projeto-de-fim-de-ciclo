# Commit Pattern — Backend ✨

Use: `<emoji>(<tipo>): <assunto>`

| Emoji |     Tipo | O que significa     | Quando alterado       |
| ----- | -------: | ------------------- | --------------------- |
| ✨    |     feat | Nova funcionalidade | All                   |
| 🐛    |      fix | Correção de bug     | All                   |
| ♻️    | refactor | Refatoração         | All                   |
| 🧪    |     test | Testes              | tests                 |
| 📦    |     deps | Dependências        | package               |
| 📝    |     docs | Documentação        | documentation, readme |
| ⚙️    |   config | Build / config      | package               |

- Use imperativo (ex.: `✨feat: add login`).
