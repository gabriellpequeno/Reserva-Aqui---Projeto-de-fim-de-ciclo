# topo da tela
- circulo com a foto do usuário 
    - caso não tenha foto usar um icone de usuario
    - Ao clicar na foto abre um meno para subir uma foto
    - abaixo nome do usuário e email
        - nome em negrito
        - email em fonte menor
    - botão de editar perfil abaixo do email
        - ao clicar abre uma tela de edição de perfil
            - semelhante a tela de cadastro
            - mas com os campos preenchidos com os dados do usuário
            - ao clicar em salvar ele deve atualizar os dados do usuário
            - ao clicar em cancelar ele deve fechar a tela

# corpo da tela

## Atividades
- Dashboard
    - abre tela de dashboard
        - capaz de ver estatísticas de reservas
        - capaz de ver estatísticas de usuários
        - capaz de ver estatísticas de quartos
        - capaz de ver estatísticas de hotéis
- Clientes (hoteis e usuários)
    - abre tela de clientes
        - capaz de buscar clientes por nome
        - capaz de buscar clientes por email
        - capaz de buscar clientes por id do cliente
        - capaz de buscar clientes por telefone
    - o clicar em um cliente ele deve abrir uma tela de dados do cliente
        - nessa tela ele pode editar os dados do cliente
        - tela semelhante a tela de cadastro
        - mas com os campos preenchidos com os dados do cliente
        - ao clicar em salvar ele deve atualizar os dados do cliente
        - ao clicar em cancelar ele deve fechar a tela
        
## Sistema
- configurações
    - abre tela de configurações
        - capaz de configurar notificações
        - abri termos de uso
        - abri termos de privacidade
        - capaz de configurar tema
        - abrir tela 'sobre'
- sair
    - ao clicar ele deve deslogar o usuário e redirecionar para a tela de login
