# observações

- Essa tela só aparece quando o usuário está logado

- São 3 variantes dessa tela:
    - usuário usuário
        - permissões:
            - ver perfil
            - editar perfil
            - ver reservas
            - cancelar reservas
            - favoritar quartos
            - sair
    - usuário anfitrião
        - permissões:
            - ver perfil
            - editar perfil
            - acesso a dashboard de anfitrião
            - ver quartos
            - adicionar quartos
            - remover quartos
            - editar quartos
            - ver reservas
            - aceitar reservas
            - recusar reservas
            - adicionar reservas
            - cancelar reservas
            - sair
    - usuário administrador
        - permissões:
            - ver perfil
            - editar perfil
            - acesso a dashboard de administrador
            - ver usuarios/anfitriões/
            - editar usuarios/anfitriões/
            - remover usuarios/anfitriões/
            - ver reservas
            - cancelar reservas
            - sair
