# Tech Stack

## Architecture
- **Model:** Client-Server
- **Communication:** RESTful API

## Backend
- **Runtime:** Node.js
- **Language:** TypeScript
- **Framework:** Express
- **Database:** PostgreSQL (via `pg` driver)
- **Security & Utilities:** `bcrypt` for password hashing, `uuid` for unique identifiers, `dotenv` for environment variables.

## Frontend
- **Framework:** Flutter
- **Language:** Dart
- **Target Platforms:** Mobile (iOS/Android) and Web

## Tooling & Infrastructure
- **Package Management:** `npm` (Backend), `flutter pub` (Frontend)
